À propos du projet

Ce projet est un portfolio interactif développé en HTML, CSS et JavaScript. Il regroupe mes projets personnels, mes compétences en développement web, et mes liens professionnels.

Le site utilise un design responsive avec un menu mobile moderne et des effets dynamiques pour les images et la navigation.

Fonctionnalités clés

  • Navigation responsive avec menu hamburger centré
  • Présentation de projets avec images, vidéos et description
  • Effet Lightbox pour afficher les images en grand
  • Bouton "retour en haut" animé
  • Section compétences avec icônes animées

Technologies utilisées

HTML5
CSS3
JavaScript
PHP (projets liés)
MySQL (projets liés)
